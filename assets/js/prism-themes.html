코드 하이라이팅 테마 비교

같은 C++ 코드를 다른 테마로 보여줍니다.

색상 규칙: 키워드(int, for, return) · 함수명 · 문자열 · 주석 · 숫자 · #include

1. One Dark (Atom 에디터)

추천

Atom 에디터의 기본 테마. 보라색 키워드 + 빨간 함수 + 초록 문자열. 가장 인기 있음.

2. VS Code Dark+ (Visual Studio Code)

추천

VS Code 기본 다크 테마. 파란 키워드 + 노란 함수 + 주황 문자열.

3. Dracula

인기 있는 보라색 기반 테마. 분홍 키워드 + 초록 문자열 + 노란 함수.

4. Material Dark

Material Design 다크 테마. 부드러운 색감.

5. Night Owl

Sarah Drasner의 인기 테마. 짙은 남색 배경 + 부드러운 색상.

6. Tomorrow (밝은 테마)

밝은 배경 테마. 블로그의 초록 톤과 잘 어울릴 수 있음.

7. Coldark Dark

차가운 톤의 다크 테마. 청색 기반.
