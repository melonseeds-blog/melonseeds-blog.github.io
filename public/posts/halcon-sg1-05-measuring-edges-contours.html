측정 계열의 큰 그림

HALCON에서 측정은 크게 세 부류로 볼 수 있다.

  1. Region 기반 측정
    area, center, orientation, bounding box 등
  2. Edge / 1D Measure 기반 측정
    선 또는 호 방향으로 edge 위치를 subpixel로 찾음
  3. XLD / Contour / Metrology 기반 측정
    subpixel contour를 추출하고 기하 형상에 fitting
실무 가이드 실무에서 정밀 치수를 다루려면 region보다 edge/contour 기반 접근이 중요하다.

1D Measuring

1D Measuring은 지정한 measurement rectangle 또는 arc 내부에서 gray profile을 분석해 edge 위치를 찾다.

2.1 기본 흐름

measure object 생성
→ image에서 edge 추출
→ edge pair 또는 single edge 선택
→ distance/position 계산
→ measure object 재사용

2.2 Measure Object

Measure object는 측정 위치, 방향, 크기, interpolation 등을 포함하는 handle이다. 반복 측정에서는 매번 새로 만들지 않고 재사용해야 한다.

중요 parameter:

  • center row/column
  • angle
  • length1 / length2
  • width
  • interpolation
  • sigma
  • threshold
  • transition
  • select

Edge Profile 이해

1D measure는 ROI 내부 gray profile에서 edge를 찾다.

3.1 Edge 방향

edge는 밝기 변화 방향에 따라 구분된다.

  • positive: dark → bright
  • negative: bright → dark
  • all: 양방향
transition 설정 주의 transition 설정이 틀리면 반대 edge를 잡거나 edge를 놓칠 수 있다.

3.2 Sigma

Sigma는 smoothing 정도이다.

  • 작음: edge 위치 민감, noise에 취약
  • 큼: noise 감소, edge blur 증가

실무적으로 sigma는 edge profile과 noise 수준에 맞춰야 한다.

noise 큼 → sigma 증가
작은 edge 간격 측정 → sigma 과대 금지

3.3 Threshold

edge amplitude 기준이다.

  • 너무 낮음: noise edge 검출
  • 너무 높음: 약한 edge 누락

threshold는 원본 gray contrast와 조명 변동을 기준으로 margin 있게 설정해야 한다.

Edge Pair 측정

두 edge 사이 거리를 측정할 때 edge pair를 사용한다.

예:

  • pin width
  • hole diameter
  • line width
  • gap
  • lead distance

4.1 Pair Selection

여러 edge가 ROI 안에 있으면 어떤 pair를 선택할지 정해야 한다.

  • first
  • last
  • strongest
  • all
  • closest
  • fuzzy measure
치수 오검사의 대표 원인 잘못된 edge pair 선택은 치수 오검사의 대표 원인이다.

Fuzzy Measuring

Fuzzy measure object는 edge 선택 조건을 더 정교하게 제어한다.

사용 이유:

  • ROI 안에 edge가 여러 개 존재
  • 단순 strongest/first 선택이 불안정
  • 예상 위치, contrast, pair size를 함께 고려해야 함

제어 가능한 feature 예:

  • contrast
  • position
  • pair position
  • size
  • size difference

실무에서는 fuzzy measure가 다음 상황에서 강하다.

  • IC lead 측정
  • 반복 패턴에서 특정 edge만 선택
  • clutter가 있는 측정
  • 예상 치수 범위가 명확한 경우

Edge Extraction

Pixel-precise 또는 subpixel edge 추출은 contour 기반 분석의 시작이다.

6.1 Edge Filter

대표적으로 Canny 계열 edge가 사용된다.

edges_sub_pix
→ XLD contour 생성

중요 parameter:

  • filter type
  • alpha/sigma
  • low threshold
  • high threshold

6.2 Subpixel Edge

subpixel edge는 pixel grid보다 더 정밀한 위치를 추정한다. 하지만 subpixel이라고 해서 실제 정확도가 자동으로 좋아지는 것은 아니다.

정확도 조건:

  • edge contrast 충분
  • blur 안정
  • noise 낮음
  • calibration 완료
  • optical distortion 보정
  • mechanical repeatability 확보

XLD Contour Processing

XLD는 HALCON에서 subpixel contour를 표현하는 중요한 object이다.

7.1 Contour 생성

  • edges_sub_pix
  • threshold 후 boundary extraction
  • model contour
  • transformed contour

7.2 Contour 선택

필요 없는 contour를 제거한다.

기준:

  • length
  • area
  • position
  • orientation
  • closed/open
  • shape

7.3 Contour 분할

복잡한 contour를 line, circle, arc 등으로 나눈다.

사용 목적:

  • 특정 edge segment만 측정
  • line fitting
  • circle fitting
  • deviation 검사

Geometric Fitting

Contour 또는 point set에 기하 형상을 fitting한다.

대표 형상:

  • line
  • circle
  • ellipse
  • rectangle
  • polygon

8.1 Line Fitting

사용 예:

  • 직선도
  • 각도
  • 평행도
  • edge 기준선

주의:

  • outlier 제거 필요
  • contour segment 선택 중요
  • edge 방향/contrast가 안정적이어야 함

8.2 Circle Fitting

사용 예:

  • hole diameter
  • circular part
  • roundness
  • center position

주의:

  • partial arc에서는 center 오차 증가
  • arc 범위가 작으면 radius 추정 불안정
  • burr/defect가 fitting에 영향

2D Metrology

HALCON의 2D metrology는 line, circle, ellipse, rectangle 등의 측정 object를 model로 관리하는 방식이다.

기본 흐름:

create metrology model
→ add metrology object
→ set parameters
→ apply metrology model
→ get results

장점:

  • 여러 측정 object를 구조화 가능
  • 반복 검사에 적합
  • alignment와 연결 가능
  • 결과 query가 체계적

주의:

  • 초기 설정이 복잡
  • edge polarity/measure length/threshold 설정 필요
  • clutter가 많은 경우 잘못된 edge 선택 가능

Calibration과 World Coordinate

정밀 측정에서 pixel distance는 최종값이 아니다.

필요한 과정:

camera calibration
→ lens distortion compensation
→ image coordinate to world coordinate
→ physical unit 측정

11.1 왜 calibration이 필요한가

  • 렌즈 왜곡
  • perspective projection
  • pixel aspect ratio
  • 카메라 설치 각도
  • working distance 변화

11.2 실무 주의

  • calibration plate와 실제 측정 plane이 일치해야 함
  • Z 높이 차이가 있으면 scale error 발생
  • telecentric lens가 아니면 높이 변화에 민감
  • calibration 후 verification target으로 검증 필요
물리 치수의 대전제 calibration 없이 물리 치수를 말하면 안 된다. pixel 거리와 mm 거리는 동일하지 않다.

핵심 정리

  • 정밀 측정은 segmentation보다 edge profile 품질이 중요하다.
  • subpixel은 조건이 맞을 때만 의미가 있다.
  • measure object는 재사용해야 성능과 구조가 좋아진다.
  • edge pair 선택 기준이 측정 안정성을 좌우한다.
  • calibration 없이 물리 치수를 말하면 안 된다.
  • XLD contour와 metrology는 중상급 HALCON 활용의 핵심이다.
중상급으로 가는 길 Region에서 XLD/Metrology로 옮겨갈 수 있는 시야가 생기면, 같은 검사 항목이라도 정밀도·반복성·유지보수성 모두에서 한 단계 위의 시스템을 설계할 수 있다.
