3D 방식 3가지 개요

산업용 3D 비전은 크게 스테레오 비전(Stereo Vision), 구조광(Structured Light), ToF(Time-of-Flight) 세 가지 방식으로 나뉜다. 각각 원리가 다르며, 정밀도, 속도, 측정 거리, 환경 내성이 크게 다르다.

그림 1. 3D 비전의 세 가지 측정 방식 원리 비교

스테레오 비전 (Stereo Vision)

스테레오 비전이란?

사람의 두 눈처럼 일정 간격(베이스라인)으로 배치된 2대의 카메라로 동일 장면을 촬영한 뒤, 좌우 이미지의 시차(Disparity)를 계산하여 깊이(Depth)를 추정하는 방식이다. 별도의 광원 없이 자연광 환경에서 동작하는 패시브(Passive) 방식이다.

장점

  • 넓은 FOV -- 대면적 측정에 유리
  • 원거리 측정 -- 수십 m 이상 가능
  • 2D 컬러 동시 취득
  • 외부 광원 간섭 없음

단점

  • 텍스처 의존 -- 무늬 없는 표면에서 실패
  • 연산량 높음 -- GPU 가속 필요
  • 정밀도 한계 -- 구조광 대비 낮음
  • 정밀 캘리브레이션 필수

구조광 (Structured Light)

구조광이란?

프로젝터로 알려진 패턴(줄무늬, 격자, 랜덤 도트 등)을 대상물에 투사하고, 카메라로 촬영한 패턴의 왜곡을 분석하여 3D 형상을 복원하는 방식이다. 라인 레이저를 사용하는 프로파일 스캐너도 구조광의 일종이다.

장점

  • 높은 정밀도 -- 수 um 급 측정 가능
  • 텍스처 무관 -- 무광/단색 표면도 OK
  • 라인 스캔 방식으로 인라인 검사 적합
  • 산업 현장 검증 완료

단점

  • 외광 간섭 -- 밝은 환경에서 패턴 소실
  • 반사/투명 물체 -- 패턴 왜곡으로 어려움
  • 측정 거리 제한 -- 보통 2m 이내
  • 이동 물체 -- 패턴 투사 시간 필요

ToF (Time-of-Flight)

ToF란?

센서에서 발사한 적외선(IR) 빛이 대상물에 반사되어 돌아오는 시간 또는 위상 차이를 측정하여 거리를 산출하는 방식이다. 각 픽셀이 독립적으로 거리를 측정하므로 한 번의 촬영으로 전체 Depth Map을 얻는다.

장점

  • 초고속 -- 수십~수백 fps 가능
  • 원거리 -- 최대 10m 이상 측정
  • 실시간 전체 Depth Map
  • 연산 부하 낮음 -- 하드웨어 기반

단점

  • 정밀도 낮음 -- mm 급 (구조광 대비)
  • 낮은 해상도 -- 보통 VGA 이하
  • 외광 간섭 -- 태양광 적외선에 취약
  • 멀티패스(Multipath) 오류 발생

정밀도별 선택

요구 정밀도에 따라 3D 방식 선택이 달라진다. 아래 기준을 참고하여 프로젝트에 맞는 방식을 선택하자.

그림 2. 정밀도 범위에 따른 3D 방식의 적용 영역
um 급 (초정밀) 구조광 -- 반도체 웨이퍼 검사, PCB 납땜 높이 측정, 정밀 금형 검사에 적합. 프린지 프로젝션 또는 고급 라인 레이저 프로파일러 사용.
0.1 mm 급 (고정밀) 구조광 또는 스테레오 -- 자동차 부품 치수 검사, 용접 비드 검사. 라인 레이저 프로파일러가 가장 보편적.
mm 급 (일반) 스테레오 또는 ToF -- 물류 체적 측정, 로봇 빈피킹, 팔레타이징. 속도와 비용 효율이 중요.
정밀도 선택 팁 측정 대상의 허용 공차(Tolerance)를 먼저 확인하라. 3D 시스템의 정밀도는 허용 공차의 1/5 이하가 권장된다. 예: 허용 공차 0.5 mm라면 시스템 정밀도는 0.1 mm 이하 필요.

속도별 선택

생산 라인의 택트 타임(Tact Time)과 컨베이어 속도가 3D 방식 선택의 핵심 기준이다.

그림 3. 속도 요구사항에 따른 3D 측정 방식
라인 레이저 프로파일러의 속도 라인 레이저 프로파일러는 프로파일 취득 속도(수 kHz)와 3D 스캔 속도를 구분해야 한다. 프로파일 취득은 빠르지만, 전체 3D 형상을 얻으려면 대상물 이동이 필요하므로 실제 3D 스캔 시간은 컨베이어 속도에 의존한다.

환경별 선택

설치 환경의 조명 조건, 온도, 진동, 분진 등이 3D 방식 선택에 큰 영향을 미친다.

환경 조건 추천 방식 주의사항
실내 (조명 제어 가능) 구조광, 스테레오, ToF 모두 가능 차폐 하우징으로 외광 차단 시 구조광 최적
실외 (태양광) 스테레오 비전 ToF/구조광은 태양광 적외선으로 간섭 발생
고온 환경 (>50C) 스테레오 비전 (원거리 설치) ToF 센서는 온도 드리프트로 정밀도 저하
진동 환경 ToF (단일 프레임) 구조광 멀티 패턴 투사 시 진동으로 정합 오류
분진/수증기 스테레오 비전 (IP67 하우징) 레이저 광원 방식은 분진으로 산란 발생
반사/광택 표면 스테레오 + 편광 필터 구조광/ToF는 정반사로 데이터 누락 발생
투명/반투명 물체 형광 구조광 또는 특수 조명 일반 3D 방식 모두 난이도 높음
멀티 3D 방식 조합 단일 방식으로 해결이 어려운 경우, 스테레오 + 구조광 조합이 효과적이다. 예를 들어 빈피킹에서 대략적 위치는 스테레오로 빠르게 파악하고, 정밀 그립 포인트는 구조광으로 측정하는 2단계 전략을 사용한다.

산업 분야별 추천

그림 4. 산업 분야별 3D 비전 방식 추천 맵
AI와 3D 비전의 결합 최근 딥러닝 기반 3D 포인트 클라우드 처리(PointNet, 3D CNN 등)가 발전하면서, 스테레오 비전의 약점(텍스처 의존성, 매칭 실패)이 크게 개선되고 있다. 특히 빈피킹, 결함 검출 분야에서 AI + 3D의 조합이 표준이 되어가고 있다.
