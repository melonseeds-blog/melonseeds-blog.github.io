포인트 클라우드란

포인트 클라우드(Point Cloud)는 3D 공간에서 물체 표면이나 환경을 나타내는 점(Point)의 집합이다. 각 점은 최소한 X, Y, Z 좌표를 가지며, 추가로 색상(RGB), 법선 벡터(Normal), 반사 강도(Intensity) 등의 속성을 포함할 수 있다.

포인트 클라우드의 핵심 특성

2D 이미지가 격자(Grid) 형태의 픽셀로 구성되는 것과 달리, 포인트 클라우드는 비정형(Unstructured) 데이터다. 점 간의 연결 관계나 순서가 없고, 밀도가 균일하지 않다. 이 특성 때문에 전용 알고리즘과 자료구조(KD-Tree, Octree)가 필요하다.

주요 라이브러리

포인트 클라우드 처리를 위한 대표적인 도구 세 가지를 비교한다. 프로젝트의 성격과 요구사항에 따라 적합한 도구가 다르다.

항목 PCL Open3D CloudCompare
언어 C++ (Python 바인딩 일부) Python / C++ GUI (C++ 기반)
난이도 높음 (빌드 복잡) 낮음 (pip install) 매우 낮음 (클릭)
성능 최고 (네이티브 C++) 좋음 (C++ 백엔드) 좋음 (대용량 지원)
알고리즘 가장 풍부 (200+) 핵심 위주 (100+) 시각화 + 플러그인
시각화 VTK 기반 (복잡) 내장 뷰어 (간편) 최고 (전문 3D 뷰어)
적합 용도 산업용 실시간 시스템 프로토타이핑, 연구 데이터 확인, 수동 분석
실무 추천 워크플로우 Open3D로 빠르게 알고리즘을 프로토타이핑하고, CloudCompare로 결과를 시각적으로 검증한 뒤, 최종 배포 시 PCL(C++)로 포팅하는 방식이 효율적이다.

전처리

3D 스캐너나 스테레오 카메라에서 획득한 원시 포인트 클라우드는 노이즈, 과도한 밀도, 이상치를 포함한다. 전처리를 통해 데이터를 정제해야 이후 알고리즘이 안정적으로 동작한다.

다운샘플링 (Voxel Grid Filter)

Voxel Grid 다운샘플링

3D 공간을 일정 크기의 복셀(Voxel, 3D 픽셀)로 분할한 뒤, 각 복셀 내의 점들을 하나의 대표점(중심점)으로 치환한다. 복셀 크기가 클수록 점이 줄어들지만 디테일도 손실된다. 일반적으로 원본의 10~30%로 축소하면서 형상은 유지한다.

그림 2. Voxel Grid 다운샘플링 - 밀집 점군을 균일한 밀도로 축소

노이즈 제거 (Outlier Removal)

Statistical Outlier Removal

각 점에서 k개 이웃까지의 평균 거리를 계산하고, 전체 평균에서 표준편차의 n배를 초과하는 점을 이상치로 제거한다.

  • 파라미터: k(이웃 수), std_ratio(표준편차 배수)
  • 장점: 글로벌 통계 기반, 안정적
  • 적합: 균일 밀도 데이터

Radius Outlier Removal

각 점을 중심으로 반경 r 내에 존재하는 이웃 점의 수를 세어, 최소 개수 미만인 점을 제거한다.

  • 파라미터: radius(반경), min_neighbors(최소 이웃 수)
  • 장점: 직관적, 국소 밀도 기반
  • 적합: 경계 영역 고립점 제거
전처리 순서가 중요하다 일반적으로 다운샘플링 -> 노이즈 제거 순서로 진행한다. 다운샘플링으로 데이터 양을 줄이면 노이즈 제거 알고리즘의 연산 시간이 크게 단축된다. 단, 극소 노이즈가 다운샘플링 결과에 영향을 줄 수 있으므로 상황에 따라 순서를 조정한다.

정합(Registration)

서로 다른 시점에서 촬영한 여러 포인트 클라우드를 하나의 좌표계로 합치는 과정을 정합(Registration)이라 한다. 3D 스캐닝, 로봇 네비게이션, 물체 인식 등에서 핵심 단계다.

ICP (Iterative Closest Point)

ICP 알고리즘

가장 널리 사용되는 정합 알고리즘. 두 포인트 클라우드(Source, Target) 간의 가장 가까운 점 쌍을 찾고, 이 쌍들 사이의 거리를 최소화하는 변환(회전 + 이동)을 반복적으로 계산한다.

그림 3. ICP 알고리즘 - 대응점 매칭과 변환 계산을 반복하여 정합

초기 정렬과 RANSAC 기반 정합

초기 정렬 필요성 ICP는 로컬 최적화 알고리즘이므로, 초기 위치가 너무 멀면 잘못된 결과에 수렴한다. 대략적인 초기 정렬(Coarse Alignment)을 먼저 수행해야 한다.
FPFH + RANSAC Fast Point Feature Histogram(FPFH) 특징점을 추출하고, RANSAC으로 대응 관계를 찾아 글로벌 정합을 수행한다. ICP의 초기 정렬로 사용.
Point-to-Plane ICP 기본 ICP(Point-to-Point) 대신 점과 평면 간 거리를 최소화하는 변형. 평면이 많은 환경에서 수렴 속도가 5~10배 빠르다.
ICP 실패 원인과 대처 오버랩(겹치는 영역)이 30% 미만이거나, 반복적 구조(평면벽, 복도)에서는 ICP가 실패할 수 있다. 이 경우 특징점 기반 정합(FPFH+RANSAC)을 선행하거나, 오버랩 비율이 높도록 스캔 경로를 조정해야 한다.

평면/형상 검출

포인트 클라우드에서 특정 기하학적 형상(평면, 실린더, 구 등)을 자동으로 검출하는 것은 산업 검사, 역설계, 환경 인식에서 핵심 작업이다.

RANSAC 평면 피팅

RANSAC (Random Sample Consensus)

무작위로 최소 점(평면의 경우 3개)을 선택하여 모델을 세우고, 나머지 점들이 이 모델에 부합하는지(Inlier/Outlier) 판별하는 과정을 반복한다. 가장 많은 Inlier를 가진 모델을 최종 결과로 채택한다. 노이즈와 이상치에 강건하다.

그림 4. RANSAC 평면 검출 - 무작위 샘플링과 반복을 통한 강건한 평면 피팅

실린더/구 검출과 클러스터링

실린더 검출 RANSAC 모델을 실린더로 확장. 7개 파라미터(중심축 방향 3 + 축 위의 점 3 + 반지름 1)를 추정. 파이프, 볼트 등 원통형 부품 검사에 활용.
구(Sphere) 검출 4개 파라미터(중심 XYZ + 반지름). 볼 베어링, 구형 부품의 위치와 크기 측정에 사용.
DBSCAN 클러스터링 밀도 기반 군집화 알고리즘. 가까운 점들을 하나의 클러스터로 묶고, 고립된 점은 노이즈로 분류. 형상 검출 전에 물체 단위로 분리할 때 유용.
형상 검출의 일반적인 파이프라인 다운샘플링 -> 노이즈 제거 -> DBSCAN 클러스터링 -> 각 클러스터에 RANSAC 적용 -> 형상 파라미터 추출. 이 순서를 따르면 복잡한 장면에서도 개별 형상을 안정적으로 검출할 수 있다.

법선 벡터 추정

법선 벡터(Normal Vector)는 각 점에서의 표면 방향을 나타낸다. 조명 계산, 메시 생성, ICP 정합, 표면 재구성 등 거의 모든 고급 처리에서 필수적으로 사용된다.

법선 벡터 추정 원리

각 점의 k개 최근접 이웃(k-NN)을 찾고, 이 이웃 점들로 공분산 행렬(Covariance Matrix)을 구성한다. 고유값 분해(PCA)를 수행하면 가장 작은 고유값에 대응하는 고유벡터가 법선 방향이 된다. 즉, 이웃 점들이 이루는 국소 평면에 수직인 방향을 법선으로 추정한다.

그림 5. 법선 벡터 추정 - k-NN 이웃으로 국소 평면을 구하고 수직 방향을 법선으로 설정
k 값 선택 가이드 k가 너무 작으면 노이즈에 민감하고, 너무 크면 세밀한 형상을 무시한다. 일반적으로 k=20~50 범위에서 시작하고, 데이터 밀도와 곡률에 따라 조정한다. Open3D에서는 estimate_normals(search_param=o3d.geometry.KDTreeSearchParamKNN(knn=30))로 설정한다.

산업 활용

포인트 클라우드 처리 기술은 제조업, 건설, 자동차, 물류 등 다양한 산업 현장에서 비접촉 3D 측정의 핵심으로 활용되고 있다.

체적 측정

적재된 원자재, 곡물, 광석 등의 부피를 포인트 클라우드로 계산. 메시 생성 후 부피 적분 또는 복셀 기반 근사.

  • 물류 창고 적재량 모니터링
  • 광산/건설 현장 토량 계산

평탄도 검사

RANSAC으로 기준 평면을 피팅하고, 각 점에서 평면까지의 거리 편차를 분석하여 평탄도를 정량화.

  • 자동차 패널 표면 검사
  • PCB 기판 휨(Warpage) 측정

갭/단차 측정

두 부품 접합면의 틈새(Gap)와 높이차(Flush)를 포인트 클라우드 단면 분석으로 측정. 자동차 바디 조립 품질 관리의 핵심.

  • 자동차 도어/펜더 갭 측정
  • 전자제품 외관 단차 검사

역설계 (Reverse Engineering)

실물 부품을 3D 스캔하여 CAD 모델로 변환. 표면 재구성(Poisson, Ball Pivoting) 후 NURBS 변환.

  • 단종 부품 복원 및 재생산
  • 금형/치공구 디지털화
산업용 3D 센서 선택 기준 체적/토량 측정처럼 넓은 영역은 LiDAR나 ToF 카메라, 평탄도/갭 측정처럼 정밀한 영역은 구조광(Structured Light) 또는 스테레오 카메라가 적합하다. 센서의 정확도(accuracy)와 정밀도(precision)를 구분하여 요구사항에 맞게 선택해야 한다.
