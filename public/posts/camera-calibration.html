캘리브레이션이란

카메라 캘리브레이션(Camera Calibration)은 3D 세계의 좌표를 2D 이미지 좌표로 매핑하는 카메라의 수학적 모델을 결정하는 과정이다. 실제 카메라는 렌즈 왜곡, 제조 공차, 센서 정렬 오차 등으로 인해 이상적인 핀홀 카메라와 차이가 있으며, 캘리브레이션을 통해 이를 보정한다.

왜 캘리브레이션이 필요한가?

3D 복원, 스테레오 비전, 로봇 비전, 정밀 계측 등 모든 응용에서 카메라 파라미터가 정확해야 결과가 신뢰할 수 있다. 캘리브레이션 없이는 이미지에서 측정한 거리나 각도가 실제와 다르며, 특히 렌즈 왜곡이 보정되지 않으면 직선이 곡선으로 보이는 등 심각한 오차가 발생한다.

그림 1. 핀홀 카메라 모델 - 3D 세계 좌표에서 2D 이미지 좌표로의 변환 과정

내부 파라미터 (Intrinsic Parameters)

내부 파라미터(Intrinsic Parameters)는 카메라 자체의 광학적, 기하학적 특성을 나타내는 값이다. 카메라가 바뀌지 않는 한 고정이며, 카메라 행렬 K로 표현된다.

K = | fx 0 cx |
      | 0 fy cy |
      | 0 0 1 |

초점거리 (fx, fy)

fx (수평 초점거리) 렌즈 중심에서 이미지 센서까지의 거리를 수평 픽셀 단위로 환산한 값. fx = f / px (f: 물리적 초점거리 mm, px: 픽셀 간격 mm)
fy (수직 초점거리) 수직 방향의 픽셀 단위 초점거리. fy = f / py. 정사각 픽셀이면 fx = fy이지만, 센서에 따라 다를 수 있다.
초점거리의 단위 렌즈 스펙에 적힌 8mm, 12mm 등은 물리적 초점거리(f)이고, 캘리브레이션으로 구하는 fx, fy는 픽셀 단위이다. 예를 들어 물리적 초점거리 12mm, 픽셀 크기 3.45um인 카메라라면 fx = 12 / 0.00345 = 약 3478 pixels.

주점 (cx, cy)

주점(Principal Point)은 렌즈 광축이 이미지 센서와 만나는 점의 픽셀 좌표이다. 이상적으로는 이미지 중심 (W/2, H/2)이지만, 실제로는 렌즈 정렬 오차로 인해 수 픽셀에서 수십 픽셀까지 차이가 날 수 있다.

왜곡 계수 (Distortion Coefficients)

내부 파라미터에 포함되는 왜곡 계수는 렌즈에 의한 이미지 왜곡을 모델링한다. OpenCV 기준으로 일반적으로 5개의 계수 (k1, k2, p1, p2, k3)를 사용한다. 왜곡에 대해서는 4장에서 자세히 다룬다.

파라미터 기호 단위 의미
수평 초점거리 fx pixel 수평 방향 픽셀 단위 초점거리
수직 초점거리 fy pixel 수직 방향 픽셀 단위 초점거리
주점 X cx pixel 광축과 센서가 만나는 수평 좌표
주점 Y cy pixel 광축과 센서가 만나는 수직 좌표
방사 왜곡 k1, k2, k3 - 렌즈 중심으로부터 거리에 따른 왜곡
접선 왜곡 p1, p2 - 렌즈-센서 비평행에 의한 왜곡

외부 파라미터 (Extrinsic Parameters)

외부 파라미터(Extrinsic Parameters)는 세계 좌표계(World Coordinate)와 카메라 좌표계(Camera Coordinate) 사이의 관계를 나타낸다. 카메라의 위치와 방향이 바뀌면 외부 파라미터도 바뀐다.

회전 행렬 R (3x3)

세계 좌표축을 카메라 좌표축으로 회전시키는 직교 행렬. 3개의 자유도(roll, pitch, yaw)를 가진다. R'R = I, det(R) = 1.

이동 벡터 t (3x1)

세계 좌표계 원점에서 카메라 좌표계 원점까지의 이동(Translation). 카메라 좌표계 기준으로 표현된 3차원 벡터 [tx, ty, tz]'.

Pc = R * Pw + t

여기서 Pw: 월드 좌표, Pc: 카메라 좌표
[R | t]는 3x4 변환 행렬
내부 vs 외부 파라미터 요약 내부 파라미터는 카메라 "자체"의 특성(렌즈, 센서)이고 한 번 구하면 고정. 외부 파라미터는 카메라의 "자세"(위치, 방향)이고 카메라를 움직이면 변한다. 캘리브레이션은 보통 내부 파라미터를 구하는 것이 주 목적이다.

렌즈 왜곡 (Lens Distortion)

실제 렌즈는 빛을 완벽하게 직선으로 통과시키지 못한다. 렌즈 왜곡은 크게 방사 왜곡(Radial Distortion)과 접선 왜곡(Tangential Distortion)으로 나뉜다.

방사 왜곡 (Radial Distortion)

렌즈 중심으로부터의 거리(r)에 따라 발생하는 왜곡. 배럴(Barrel) 왜곡과 핀쿠션(Pincushion) 왜곡이 있다.

그림 2. 방사 왜곡 - 배럴 왜곡(볼록)과 핀쿠션 왜곡(오목) 비교
x_distorted = x(1 + k1*r2 + k2*r4 + k3*r6)
y_distorted = y(1 + k1*r2 + k2*r4 + k3*r6)

r2 = x2 + y2 (렌즈 중심으로부터의 거리)

접선 왜곡 (Tangential Distortion)

렌즈가 이미지 센서와 완벽하게 평행하지 않을 때 발생하는 왜곡이다. 렌즈 조립 공정에서의 미세한 기울어짐이 원인이며, 이미지가 한쪽으로 치우치는 형태로 나타난다.

x_distorted = x + [2*p1*x*y + p2*(r2 + 2*x2)]
y_distorted = y + [p1*(r2 + 2*y2) + 2*p2*x*y]
왜곡 보정 순서에 주의 왜곡 보정(Undistortion)은 반드시 다른 이미지 처리(에지 검출, 매칭 등) 이전에 수행해야 한다. 왜곡된 이미지에서 추출한 특징점은 위치가 부정확하여 후속 처리의 정확도를 떨어뜨린다.

체커보드 캘리브레이션

체커보드 패턴은 가장 널리 사용되는 캘리브레이션 타겟이다. 흑백 격자의 코너점을 서브픽셀 정밀도로 검출할 수 있어 높은 정확도를 달성할 수 있다.

그림 3. 체커보드 캘리브레이션 절차 및 코너점 검출 예시
OpenCV 캘리브레이션 함수

OpenCV에서는 cv::findChessboardCorners()로 코너를 검출하고, cv::cornerSubPix()로 서브픽셀 보정을 수행한 후, cv::calibrateCamera()로 내부/외부 파라미터를 한번에 추정한다. 결과로 카메라 행렬 K, 왜곡 계수, 각 이미지별 R과 t가 반환된다.

체커보드 외 캘리브레이션 타겟 체커보드 외에도 원형 패턴(Circle Grid), ChArUco 보드(체커보드 + ArUco 마커), Halcon 캘리브레이션 플레이트 등이 있다. ChArUco는 부분적으로 가려져도 검출이 가능하여 자동화 환경에서 유리하다.

스테레오 캘리브레이션

스테레오 캘리브레이션은 두 카메라 사이의 상대적 위치 관계(R, T)를 정밀하게 구하는 과정이다. 각 카메라의 내부 파라미터를 먼저 구한 후, 동일한 체커보드를 두 카메라로 동시에 촬영하여 상대 자세를 추정한다.

그림 4. 스테레오 카메라 기하학 - 두 카메라 간의 R, T와 삼각측량 원리

스테레오 캘리브레이션 절차

1. 개별 캘리브레이션 좌/우 카메라 각각 단독으로 캘리브레이션하여 내부 파라미터(K, D)를 구한다.
2. 스테레오 촬영 동일한 체커보드를 두 카메라로 동시에 촬영한다. 최소 15~20쌍의 이미지가 필요하다.
3. stereoCalibrate OpenCV의 cv::stereoCalibrate()로 두 카메라 간의 R(회전), T(이동)를 추정한다.
4. stereoRectify cv::stereoRectify()로 두 이미지를 수평 정렬(Rectification)하여 에피폴라 라인을 수평으로 만든다.
5. Remap 생성 cv::initUndistortRectifyMap()으로 왜곡 보정 + 정렬 변환을 합친 Remap 테이블을 생성한다.
Rectification의 중요성 스테레오 정렬(Rectification)을 수행하면 좌/우 이미지에서 같은 3D 점이 같은 y좌표(행)에 나타나게 된다. 이를 통해 대응점 탐색을 1차원(수평 방향)으로 줄일 수 있어 스테레오 매칭 속도와 정확도가 크게 향상된다.

정확도 검증

캘리브레이션 결과의 신뢰성을 확인하는 것은 실제 응용에서 매우 중요하다. 주요 검증 방법과 판단 기준을 알아본다.

재투영 오차 (Reprojection Error)

캘리브레이션으로 구한 파라미터를 사용하여 3D 점을 다시 이미지에 투영했을 때, 실제 검출된 코너점과의 거리를 측정한다. 가장 기본적이고 중요한 지표이다.

재투영 오차 (RMS) 평가 비고
< 0.3 pixel 우수 정밀 계측, 스테레오 비전에 적합
0.3 ~ 0.5 pixel 양호 일반적인 머신비전 응용에 충분
0.5 ~ 1.0 pixel 보통 정밀도가 낮은 응용에서는 사용 가능
> 1.0 pixel 불량 재캘리브레이션 필요. 이미지 품질 확인

에피폴라 오차 (스테레오)

스테레오 캘리브레이션에서는 대응점이 에피폴라 라인 위에 정확히 놓이는지 확인한다. 정렬(Rectification) 후 좌/우 이미지의 대응점은 같은 행(y좌표)에 있어야 하며, y좌표 차이가 에피폴라 오차이다.

좋은 캘리브레이션을 위한 체크리스트 1) 재투영 오차 RMS < 0.5 pixel 이하인가?
2) 왜곡 보정 후 직선이 직선으로 보이는가?
3) (스테레오) 정렬 후 에피폴라 라인이 수평인가?
4) 이미지 전체 영역에서 오차가 균일한가? (특정 영역에 오차 집중 시 재촬영)
5) 이상치(Outlier) 이미지를 제거하고 재캘리브레이션했는가?

실전 검증 방법

실측 비교 검증

캘리브레이션 결과를 실제 환경에서 검증하는 가장 확실한 방법이다. 알려진 크기의 물체(게이지 블록, 자)를 촬영하고, 이미지에서 측정한 치수와 실제 치수를 비교한다. 스테레오 비전의 경우 알려진 거리에 놓인 물체의 깊이를 추정하여 오차를 확인한다.

오차가 클 때 확인할 점

  • 체커보드 평탄도 (휘어짐 확인)
  • 촬영 시 포커스 정확성
  • 조명 반사/글레어 여부
  • 이미지 블러 (흔들림)
  • 격자 크기 입력값 정확성

정확도 향상 팁

  • 더 많은 이미지 촬영 (30장 이상)
  • 이미지 전 영역에 체커보드 분포
  • 다양한 거리에서 촬영
  • 고정밀 캘리브레이션 플레이트 사용
  • 이상치 이미지 수동 제거
