구조광이란

구조광(Structured Light)은 알려진 패턴(줄무늬, 격자, 점 등)을 대상 물체에 투사하고, 카메라로 관찰한 패턴의 변형(왜곡)을 분석하여 3D 형상을 복원하는 기술이다. 평평한 면에서는 패턴이 그대로 보이지만, 높낮이가 있는 표면에서는 패턴이 휘어지거나 이동한다. 이 변형량을 정밀하게 측정하면 각 픽셀의 높이(depth)를 계산할 수 있다.

핵심 아이디어

프로젝터가 투사한 패턴은 "구조(structure)"가 이미 알려져 있다. 카메라가 촬영한 영상에서 이 구조가 어떻게 변형되었는지를 역산하면, 프로젝터-카메라-물체 사이의 삼각측량(triangulation)을 통해 3D 좌표를 얻는다. 스테레오 비전이 두 카메라를 쓰는 것과 달리, 구조광은 프로젝터가 한쪽 "눈" 역할을 대신한다.

패턴 종류

구조광에서 투사하는 패턴의 설계는 측정 정밀도, 속도, 노이즈 내성에 직접적인 영향을 준다. 대표적인 패턴 유형을 살펴보자.

바이너리 코드 패턴

가장 직관적인 방법으로, 흑백 줄무늬의 폭을 점점 절반으로 줄여가며 여러 장의 패턴을 순차 투사한다. 각 픽셀 위치는 밝음(1)/어두움(0)의 조합으로 고유한 이진 코드를 부여받는다. N장의 패턴으로 2^N개의 고유 위치를 구분할 수 있다.

그림 2. 바이너리 코드 패턴 - 순차적으로 투사하여 고유 코드 부여

그레이 코드 패턴

바이너리 코드의 개선형으로, 인접 영역 간 코드가 1비트만 달라지도록 설계한다. 바이너리 코드에서는 경계 부근에서 여러 비트가 동시에 바뀌어 디코딩 오류가 발생하기 쉽지만, 그레이 코드는 경계에서 1비트만 전환되므로 에러에 강하다.

그레이 코드 vs 바이너리 코드 바이너리 코드 3 -> 4는 011 -> 100으로 3비트가 동시에 바뀐다. 경계에서 약간의 노이즈나 블러만 있어도 111, 010 등 잘못된 코드가 읽힐 수 있다. 그레이 코드는 010 -> 110처럼 항상 1비트만 바뀌어 경계 오류를 최소화한다.

사인파(정현파) 패턴

밝기가 사인 함수 형태로 부드럽게 변하는 줄무늬를 투사한다. 바이너리/그레이 코드가 영역 단위로 이산적(discrete)인 반면, 사인파 패턴은 연속적인 밝기 변화를 이용하여 서브픽셀 수준의 정밀도를 달성한다. 위상 이동법(Phase Shifting)의 핵심 패턴이다.

그림 3. 사인파 패턴 - 연속적 밝기 변화로 서브픽셀 정밀도 확보

위상 이동법 (Phase Shifting)

위상 이동법은 동일한 사인파 패턴의 위상을 일정 간격으로 이동(shift)하며 여러 장을 촬영한 뒤, 각 픽셀의 위상(phase)을 정밀하게 계산하는 기법이다. 가장 널리 쓰이는 방법은 3단계 또는 4단계 위상 이동이다.

3단계 / 4단계 위상 이동

위상 이동 공식

3단계 위상 이동: 위상을 0, 2pi/3, 4pi/3 만큼 이동한 3장을 촬영하여 각 픽셀의 위상을 계산한다.

phi(x,y) = arctan[ sqrt(3) * (I1 - I3) / (2*I2 - I1 - I3) ]

4단계 위상 이동: 위상을 0, pi/2, pi, 3pi/2 만큼 이동한 4장을 촬영한다. 3단계보다 노이즈에 강하다.

phi(x,y) = arctan[ (I4 - I2) / (I1 - I3) ]

그림 4. 4단계 위상 이동법 - 4장의 영상에서 각 픽셀의 위상을 계산

위상 언래핑과 높이 변환

arctan 함수의 결과는 -pi ~ pi 범위의 래핑된(wrapped) 위상이다. 실제 3D 형상을 복원하려면 이 위상을 연속적인 값으로 언래핑(unwrapping)해야 한다. 이후 언래핑된 위상을 시스템의 기하학적 파라미터를 이용하여 실제 높이(Z)로 변환한다.

위상 모호성(Phase Ambiguity) 사인파 패턴은 주기적이므로, 하나의 위상값만으로는 어느 주기에 해당하는지 알 수 없다. 이를 해결하기 위해 서로 다른 주파수의 패턴을 추가 투사하거나, 그레이 코드와 결합하여 절대 위상(absolute phase)을 결정한다.

코드화 패턴

구조광 패턴의 코딩 방식은 크게 시간 코딩(Temporal Coding)과 공간 코딩(Spatial Coding)으로 분류되며, 두 방식을 결합한 하이브리드 기법도 널리 사용된다.

시간 코딩 vs 공간 코딩

시간 코딩 (Temporal)

  • 여러 장의 패턴을 순차적으로 투사
  • 각 픽셀이 시간에 걸쳐 고유 코드를 형성
  • 바이너리/그레이 코드, 위상 이동법이 대표적
  • 장점: 높은 정밀도, 큰 해상도
  • 단점: 여러 장 촬영 필요 (움직이는 물체에 부적합)

공간 코딩 (Spatial)

  • 한 장의 패턴만으로 코딩
  • 각 픽셀의 이웃 영역(window)을 분석하여 코드 결정
  • 컬러 스트라이프, De Bruijn 시퀀스가 대표적
  • 장점: 단일 촬영 (실시간, 동적 물체 가능)
  • 단점: 정밀도가 시간 코딩보다 낮음

하이브리드 기법

실제 산업용 3D 스캐너는 시간 코딩과 공간 코딩의 장점을 결합한 하이브리드 방식을 많이 사용한다. 예를 들어:

그레이 코드 + 위상 이동 그레이 코드로 절대 위치(coarse)를 결정하고, 사인파 위상 이동으로 서브픽셀 정밀도(fine)를 보완. 가장 널리 쓰이는 조합.
다중 주파수 서로 다른 주파수의 사인파를 순차 투사하여 위상 모호성을 제거. 고해상도 패턴일수록 정밀하지만 모호성이 커지므로, 저주파로 절대 위치를 잡는다.
컬러 + 시간 코딩 컬러 패턴으로 공간 코딩 효과를 얻으면서, 추가 프레임으로 시간 코딩 정밀도 보완. 패턴 수를 줄여 속도 향상.

구조광 시스템 구성

구조광 3D 스캐닝 시스템은 크게 프로젝터, 카메라, 그리고 이 둘의 기하학적 관계를 정의하는 캘리브레이션으로 구성된다.

프로젝터 + 카메라 구성

프로젝터 DLP(Digital Light Processing) 또는 LCoS 방식. 고속 패턴 전환이 중요하며, 산업용은 수 kHz 속도로 패턴을 투사한다. LED 또는 레이저 광원 사용.
카메라 글로벌 셔터 산업용 카메라. 프로젝터와 정확한 동기(trigger) 필요. 해상도는 측정 분해능에 직접 영향.
배치 (Baseline) 프로젝터와 카메라 사이의 거리(Baseline)가 클수록 깊이 분해능이 높아지지만, 그림자(occlusion)가 증가하는 트레이드오프가 있다.

캘리브레이션

정확한 3D 복원을 위해 카메라의 내부 파라미터(초점거리, 렌즈 왜곡), 프로젝터의 내부 파라미터, 그리고 카메라-프로젝터 간의 외부 파라미터(상대 위치/회전)를 정밀하게 측정해야 한다.

프로젝터도 "역카메라" 캘리브레이션 관점에서 프로젝터는 빛을 쏘는 "역(inverse) 카메라"로 모델링할 수 있다. 체커보드 패턴을 카메라로 촬영하면서 동시에 프로젝터로 패턴을 투사하면, 두 장치의 대응 관계를 구할 수 있다.

삼각측량 원리

그림 6. 삼각측량 - 프로젝터 투사각과 카메라 관측각에서 깊이 Z를 계산

장단점

장점

  • 높은 정밀도 - 위상 이동법 적용 시 수 마이크로미터 분해능
  • 텍스처 무관 - 스테레오 비전과 달리 무늬 없는 표면도 측정 가능
  • Dense 3D - 전 영역에 대해 밀도 높은 3D 포인트 클라우드 생성
  • 비접촉 - 물리적 접촉 없이 측정하여 대상 손상 없음
  • 넓은 측정 범위 - 수 mm에서 수 m까지 스케일 조절 가능

단점

  • 외광 영향 - 주변광(ambient light)이 강하면 패턴 대비 저하
  • 촬영 시간 - 시간 코딩은 여러 장 필요, 움직이는 물체에 제한
  • 그림자/가림 - 프로젝터-카메라 배치에 따라 사각 영역 발생
  • 반사면 문제 - 경면 반사(specular)가 심한 금속 표면은 측정 어려움
  • 시스템 비용 - 고속 프로젝터와 정밀 캘리브레이션 필요

산업 적용

PCB 3D 검사 (SPI)

SPI (Solder Paste Inspection)

PCB 기판 위에 도포된 솔더 페이스트(납땜용 크림)의 체적, 높이, 면적을 3D로 정밀 측정하는 장비. 구조광의 대표적 산업 응용 사례이다. 위상 이동법으로 수 마이크로미터 정밀도를 달성하며, Moire 기법이나 컬러 패턴도 활용된다.

SPI 장비 주요 사양 대표적인 SPI 장비(KohYoung, CyberOptics 등)는 Z축 분해능 0.1~1 um, XY 분해능 10~20 um, 측정 속도 수십 cm^2/s 수준이다. SMT 라인에서 리플로우 전 100% 검사에 사용된다.

치과 스캐너

구강 내부를 실시간으로 3D 스캔하여 치아 모형을 디지털화한다. 환자의 구강에 소형 프로젝터+카메라 모듈을 넣고 구조광 패턴을 투사하여 치아 형상을 취득한다. 인상재(몰드) 없이 크라운, 인레이, 교정 장치를 제작할 수 있어 치과 디지털 워크플로의 핵심 장비이다.

정밀도 10~30 um 수준. 보철물 제작에 충분한 정밀도를 제공한다.
대표 제품 3Shape TRIOS, Dentsply Sirona CEREC, Align iTero 등
투사 방식 소형 DLP + LED. 컬러 패턴 또는 고속 위상 이동법 적용.

로봇 비전 가이드

산업용 로봇이 부품을 집거나(bin picking) 조립할 때, 구조광 3D 카메라로 대상의 위치와 자세를 파악한다. 텍스처가 없는 금속 부품도 측정할 수 있어 스테레오 비전보다 유리하다.

Bin Picking

  • 상자 안에 무질서하게 쌓인 부품의 3D 형상 취득
  • 포인트 클라우드에서 개별 부품 인식/자세 추정
  • 로봇 그리퍼가 정확한 위치를 잡고 픽업
  • 대표 제품: Photoneo, Zivid, Ensenso

3D 품질 검사

  • 부품의 치수, 평탄도, 갭/단차를 실시간 측정
  • CAD 모델과 비교하여 OK/NG 자동 판정
  • 자동차 차체, 전자부품, 금형 검사 등
  • 인라인(라인 내) 전수검사에 적용
실시간 구조광의 발전 최근에는 고속 DLP 프로젝터(수만 Hz 패턴 전환)와 GPU 기반 실시간 처리 기술의 발전으로, 구조광 방식도 30fps 이상의 실시간 3D 스캐닝이 가능해지고 있다. 이는 컨베이어 위를 이동하는 물체의 인라인 3D 검사나 로봇 실시간 가이드에 중요하다.
