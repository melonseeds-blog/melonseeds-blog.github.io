STL이란?

Standard Template Library의 약자로, C++ 표준 라이브러리의 핵심 구성 요소다. 자주 사용하는 자료구조와 알고리즘을 템플릿으로 제공하여 직접 구현할 필요 없이 바로 사용할 수 있다.

구성 요소역할예시
컨테이너데이터를 저장하는 자료구조vector, map, set
반복자컨테이너의 요소를 순회begin(), end()
알고리즘정렬, 탐색 등 범용 함수sort(), find()

주요 컨테이너

vector — 동적 배열

가장 자주 쓰는 컨테이너. 크기가 자동으로 늘어나는 배열이다.

#include <vector>

vector<int> v;              // 빈 벡터
vector<int> v(10, 0);       // 크기 10, 초기값 0
vector<int> v = {1, 2, 3};  // 초기화 리스트

v.push_back(4);       // 끝에 추가
v.pop_back();         // 끝에서 제거
v.size();             // 크기
v.empty();            // 비었는지
v[0];                 // 인덱스 접근 (범위 체크 X)
v.at(0);              // 인덱스 접근 (범위 체크 O, 느림)
v.front();            // 첫 번째 요소
v.back();             // 마지막 요소
v.clear();            // 전체 삭제
v.resize(20);         // 크기 변경
주의: vector<bool>은 특수하다 비트 단위로 저장되어 &v[0] 같은 주소 참조가 안 된다. vector<char>나 deque<bool>을 대신 사용하자.

map — 키-값 쌍 (정렬됨)

Red-Black Tree 기반. 키가 자동 정렬된다. 탐색/삽입/삭제 O(log n).

#include <map>

map<string, int> m;
m["apple"] = 3;           // 삽입/수정
m.insert({"banana", 5});  // 삽입

// 키 존재 확인
if (m.find("apple") != m.end()) { /* 찾음 */ }
if (m.count("apple") > 0) { /* 찾음 */ }  // 0 또는 1

// 순회 (키 순서대로)
for (auto& [key, val] : m) {
    cout << key << ": " << val << endl;
}

m.erase("apple");  // 키로 삭제
m.size();           // 크기

unordered_map — 해시맵

해시 테이블 기반. 정렬 안 됨. 평균 O(1) 탐색. map보다 빠르지만 순서 보장 안 됨.

#include <unordered_map>

unordered_map<string, int> um;
um["key"] = 100;
// 사용법은 map과 동일. 순회 시 순서만 보장 안 됨.
map vs unordered_map 언제 쓸까? 키 순서가 필요하면 map, 빠른 탐색만 필요하면 unordered_map. 대부분의 경우 unordered_map이 더 빠르다.
비교mapunordered_map
내부 구조Red-Black TreeHash Table
탐색O(log n)O(1) 평균
순서키 정렬순서 없음
사용 시점정렬 필요할 때빠른 탐색이 중요할 때

set — 중복 없는 집합

map에서 값을 뺀 것. 키만 저장. 자동 정렬, 중복 불가.

#include <set>

set<int> s;
s.insert(3);
s.insert(1);
s.insert(3);  // 무시됨 (중복)
// s = {1, 3}

s.erase(1);
s.count(3);   // 0 또는 1
s.find(3);    // iterator 반환

stack, queue, priority_queue

#include <stack>
#include <queue>

// Stack (LIFO — Last In First Out)
stack<int> st;
st.push(1);
st.top();     // 맨 위 확인
st.pop();     // 맨 위 제거

// Queue (FIFO — First In First Out)
queue<int> q;
q.push(1);
q.front();    // 맨 앞
q.back();     // 맨 뒤
q.pop();      // 맨 앞 제거

// Priority Queue (최대 힙)
priority_queue<int> pq;
pq.push(3);
pq.push(1);
pq.push(5);
pq.top();     // 5 (가장 큰 값)
최소 힙 만들기 priority_queue<int, vector<int>, greater<int>>으로 선언하면 top()이 가장 작은 값을 반환한다.

deque — 양방향 큐

#include <deque>

deque<int> dq;
dq.push_back(1);   // 뒤에 추가
dq.push_front(2);  // 앞에 추가
dq.pop_back();     // 뒤에서 제거
dq.pop_front();    // 앞에서 제거
dq[0];             // 인덱스 접근 가능

반복자 (Iterator)

컨테이너의 요소를 가리키는 포인터처럼 동작한다.

vector<int> v = {10, 20, 30};

// 기본 순회
for (auto it = v.begin(); it != v.end(); ++it) {
    cout << *it << endl;
}

// range-based for (C++11, 가장 간결)
for (auto& x : v) {
    cout << x << endl;
}

// 역순회
for (auto it = v.rbegin(); it != v.rend(); ++it) {
    cout << *it << endl;
}
실행 결과 (range-based for)
10 20 30
실무에서는 range-based for를 쓰자 for (auto& x : v)가 가장 간결하고 실수가 적다. 인덱스가 필요한 경우에만 일반 for를 사용.

주요 알고리즘

#include <algorithm> 필요.

정렬

vector<int> v = {3, 1, 4, 1, 5};

sort(v.begin(), v.end());                 // 오름차순
sort(v.begin(), v.end(), greater<int>()); // 내림차순

// 커스텀 정렬 (람다)
sort(v.begin(), v.end(), [](int a, int b) {
    return abs(a) < abs(b);  // 절대값 기준 오름차순
});

탐색

vector<int> v = {1, 3, 5, 7, 9};

// 선형 탐색 O(n)
auto it = find(v.begin(), v.end(), 5);
if (it != v.end()) { /* 찾음 */ }

// 이진 탐색 O(log n) — 정렬된 상태에서만!
bool found = binary_search(v.begin(), v.end(), 5);

// lower_bound: 값 이상인 첫 위치
auto lb = lower_bound(v.begin(), v.end(), 4);  // → 5를 가리킴

// upper_bound: 값 초과인 첫 위치
auto ub = upper_bound(v.begin(), v.end(), 5);  // → 7을 가리킴
binary_search는 반드시 정렬된 컨테이너에서만 사용 정렬 안 된 상태에서 사용하면 잘못된 결과가 나온다. sort() 후 사용하거나, 처음부터 정렬된 set/map을 쓰자.

기타 유용한 알고리즘

// 최대/최소
*max_element(v.begin(), v.end());
*min_element(v.begin(), v.end());

// 합계
#include <numeric>
int sum = accumulate(v.begin(), v.end(), 0);

// 중복 제거 (정렬 후)
sort(v.begin(), v.end());
v.erase(unique(v.begin(), v.end()), v.end());

// 뒤집기
reverse(v.begin(), v.end());

// 순열
next_permutation(v.begin(), v.end());

// 개수 세기
int cnt = count(v.begin(), v.end(), 3);
int cnt2 = count_if(v.begin(), v.end(), [](int x) { return x > 3; });

string 관련

#include <string>

string s = "hello world";

s.length();             // 11
s.substr(0, 5);         // "hello"
s.find("world");        // 6 (인덱스), 못 찾으면 string::npos
s.replace(0, 5, "hi");  // "hi world"

// 문자열 → 숫자
int n = stoi("123");
double d = stod("3.14");

// 숫자 → 문자열
string ns = to_string(123);

pair와 tuple

#include <utility>  // pair
#include <tuple>    // tuple

// pair — 2개 묶기
pair<int, string> p = {1, "one"};
p.first;   // 1
p.second;  // "one"
auto p2 = make_pair(2, "two");

// tuple — 3개 이상 묶기
tuple<int, string, double> t = {1, "one", 1.0};
get<0>(t);  // 1
get<1>(t);  // "one"
auto [a, b, c] = t;  // 구조화 바인딩 (C++17)
